# 🚗 Used Car Price Analysis Using Python & Pandas

## 📌 Project Overview
This project performs Exploratory Data Analysis (EDA) on a used car dataset to identify factors affecting vehicle selling prices.

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📊 Key Insights
- Newer cars have higher selling prices.
- Higher mileage vehicles have lower resale values.
- Vehicle condition significantly affects price.
- Some brands command premium prices.

## 📈 Visualizations
- Bar Charts
- Line Charts
- Donut Charts
- Heatmaps
- Box Plots
