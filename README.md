# Car_listing
🚗 Used Car Price Analysis Using Python & Pandas
📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on a used car dataset containing information such as car brand, model, year, condition, mileage, color, state, and selling price.

The goal is to clean the data, analyze key factors affecting car prices, and generate meaningful business insights using Python.

🎯 Objectives
Perform data cleaning and preprocessing
Handle missing values and duplicates
Analyze selling prices across different features
Create visualizations to identify trends and patterns
Extract business insights from the dataset
🛠️ Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Google Colab / Jupyter Notebook

State-wise Car Sales Analysis
Correlation Analysis
Outlier Detection & Removal
Data Visualization
🔍 Key Insights
🚗 Newer cars generally have higher selling prices.
📉 Higher mileage vehicles tend to have lower resale values.
💰 Vehicle condition significantly impacts selling price.
🏆 Certain brands command higher average prices.
🌎 Car sales are concentrated in a few states.
📊 Year, condition, and odometer are strong factors influencing price.
